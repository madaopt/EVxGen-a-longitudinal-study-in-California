# Overview

This repository contains the code used in the study:

**“Electric Vehicle Adoption, Gentrification, and Housing Prices: A
Longitudinal Study in California.”**

------------------------------------------------------------------------

# Data Sources

The analysis integrates multiple datasets, including:

- California Air Resources Board (CARB) EV data:
  <https://arb.ca.gov/emfac/fleet-db/71921f2ab22a4a3ce61b63060283eced83e5ed16%20>
- ACS demographic and housing price data from IPUMS NHGIS: Jonathan
  Schroeder, David Van Riper, Steven Manson, Katherine Knowles, Tracy
  Kugler, Finn Roberts, and Steven Ruggles. IPUMS National Historical
  Geographic Information System: Version 20.0 \[dataset\]. Minneapolis,
  MN: IPUMS. 2025. <http://doi.org/10.18128/D050.V20.0>
- UDP gentrification typology:
  <https://github.com/urban-displacement/displacement-typologies>
- CalEnviroScreen 4.0:
  <https://oehha.ca.gov/calenviroscreen/report/calenviroscreen-40>
- Lastly, making sure crosswalk data from 2020 universe to 2010 for this
  analysis: <https://www.nhgis.org/geographic-crosswalks>

Note: Due to privacy restrictions, raw data are not publicly distributed
in this repository, but they can be freely downloaded through the above
URLs.
